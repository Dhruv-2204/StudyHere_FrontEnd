# StudyHere_FrontEnd

## Project Description
Vue.js frontend application for an after-school classes booking system. This application allows students and parents to browse, search, and book various after-school classes and activities.

## Technology Stack
- **Framework:** Vue.js 2 
- **Styling:** TAILWIND CSS 
- **HTTP Client:** Native Fetch API
- **Routing:** Vue Router
- **Icons:** Font Awesome
- **Deployment:** GitHub Pages


## Features Implemented
-  Display list of lessons with images and icons
-  Sort lessons by subject, location, price, and availability
-  Add lessons to shopping cart
-  Shopping cart management with remove functionality
-  Checkout form with validation (name: letters only, phone: numbers only)
-  Full-text search/search as you type functionality
-  Responsive design
